<br>文 / 西打藍 Siddharam<br><br>

這週五，<br>
我走在台北路上，<br>
突然創了新 IG 帳號。<br><br>

在沒有規劃下，<br>
剪輯一支當天拍的影片，<br>
直接就發了。<br><br>

隔一天，<br>
我又拍了一部，<br>
手機剪完就發。<br><br>

我為什麼這麼做？<br>
答案是：「拋下包袱。」<br><br>

而這正是經營自媒體的人，<br>
會卡住的地方。<br><br>


<h3 class="article-h1-color">▎你，想要表現很好</h3><br>

3 年前，<br>
我玩 IG 的原因，<br>
就是想換個地方，<br>
放長文章。<br><br>

我並沒有好好玩 IG，<br>
就只是放長文，<br>
並逐步學習各種「經營」方法。<br><br>

後來有短影音了，<br>
我仍用一種「專業方式」玩它。<br><br>

每次發影片都準備好久，<br>
後來停更。<br>
這些都源於一個原因：<br><br>

「想要表現更好。」<br><br>

想經營自媒體的你，<br>
一定曾有相似感受：<br><br>


<h3 class="article-h1-color">▎你為什麼拖延？</h3><br>

首先，<br>
你會花很久時間想題材；<br><br>

接著，<br>
你開始拍攝、寫文，<br>
花力氣修修改改；<br><br>

發文後，<br>
你時刻關心數據，<br>
想著「讀者喜歡嗎？」<br><br>

這些種種，<br>
導致我們超久發一次文。<br>
甚至因為成效不好，<br>
就不再發了。<br><br>

我花了很大努力，<br>
已經最簡化發文方式，<br>
但仍然很有包袱。<br><br>

突然有天，<br>
我看見解決問題的方式。<br><br>


<h3 class="article-h1-color">▎改變我的一部影片</h3><br>

前幾天滑手機時，<br>
看到一部影片，<br>
分享一種記錄生活方式：<br><br>

時間 + 心情 + 景色 + 當時感受<br><br>

影片還補充說：<br>
「不追求好看，只紀錄當下。」<br><br>

我整個被打動，<br>
超想馬上拍一部。<br>
我當下沒意識到，<br>
這能解決我的問題。<br><br>


<h3 class="article-h1-color">▎突然營業</h3><br>

週五，<br>
我去整理永和區的房間，<br>
準備帶房客看房。<br><br>

我躺在小沙發，<br>
不小心睡著。<br><br>

起床後，<br>
想到那部影片，<br>
馬上就拍了一段。<br><br>

接著沿路忙到晚上，<br>
在回家路上想著：<br><br>

「拍都拍了，不發好可惜。」<br>
但，要發在哪裡呢？<br><br>


<h3 class="article-h1-color">▎我創了影片帳號</h3><br>

我原本想發在原本帳號，<br>
但始終有點糾結，<br>
覺得貼文調性差太多。<br><br>

於是我心想：<br>
「乾脆重來一次吧。」<br>
新帳號就誕生了。<br><br>

我用手機軟體上字幕，<br>
簡單做封面，<br>
約半小時，就發文了。<br><br>

一切都沒有規劃，<br>
突然就做了。<br>
我也感受到一股很不一樣的心情。<br><br>


<h3 class="article-h1-color">▎重新玩一次</h3><br>

我發文後，<br>
重複看好多次。<br>
想著：<br><br>

「明年回頭看，一定會有滿滿感動。」<br><br>

因為有分享限動關係，<br>
幾位老朋友來留言、追蹤，<br>
還聊了一下天。<br><br>

我突然有一種，<br>
創了個人帳號，<br>
重新玩 IG 的感覺。<br><br>

接著，我又做了一部影片。<br><br>


<h3 class="article-h1-color">▎和朋友的某一刻</h3><br>

隔天，<br>
我跟夥伴帶租客看房，<br>
當下想著：<br><br>

「跟朋友拍影片紀錄當下，<br>
不就是我很想做的事嗎？」<br><br>

於是第二部影片誕生了，<br>
介紹了超常碰面的工作夥伴。<br><br>

我同時也想到超多可能性，<br>
充滿期待。<br><br>


<h3 class="article-h1-color">▎我的新實驗場</h3><br>

我開始想：<br><br>

有沒有想跟誰拍？<br>
是不是可以直播看看？<br>
要不要準備什麼造型呢？<br><br>

我開始有點期待每一天，<br>
有哪些事想記錄下來。<br><br>

同時我也思考，<br>
對比以前的發文流程，<br>
少了好多「拖延」。<br><br>


<h3 class="article-h1-color">▎終於能輕鬆發文了</h3><br>

我不用思考主題，<br>
紀錄當下就好；<br><br>

我也不用準備完美，<br>
上個字幕，做個封面，<br>
就可以發了；<br><br>

新帳號，<br>
也沒什麼數據好看，<br>
反而有很多實驗可能性。<br><br>

那麼，<br>
對於想經營 IG 的你，<br>
我想分享一些心得。<br><br>


<h3 class="article-h1-color">▎放下經營</h3><br>

你不一定要按照我的格式做，<br>
但可以用「新流程」，<br>
解決現況問題。<br><br>

如果你花很多時間想標題、<br>
做內容、修修改改，<br>
那麼你該思考：<br><br>

「有什麼新作法，<br>
能讓你穩定發文，減少拖延？」<br><br>

自媒體的精髓，<br>
就是持續。<br><br>

只要持續做，<br>
你就會慢慢進步。<br><br>

最怕的，<br>
是規劃太完整，<br>
結果行動力超級差。<br><br>


<h3 class="article-h1-color">▎紀錄真實生活</h3><br>

我的相簿裡，<br>
總有好多照片、影片，<br>
不知道該怎麼整理。<br><br>

如果有一個地方，<br>
可以紀錄最想記得的那一刻，<br>
好像就不用存那麼多照片了。<br><br>

新的帳號，<br>
讓我有從零玩 IG 的感覺。<br><br>

這或許也是社群軟體最初的存在意義：<br>
「交朋友，紀錄生活。」<br><br>

謝謝那部影片，<br>
讓我重新玩一次社群。<br><br>
