<h3 class="article-h1-color">無聊</h3><br>


請針對「吃素的人，為什麼就往往喜歡跟別人說他吃素這件事？」這個問題，寫下一個你認為最合理的原因：<br><br>

椅子是什麼？我可以用一個詞回答問題，就是概念。<br><br>

無聊是什麼？<br><br>

很閒、很忙，是用時間衡量，但不會說他的本質是時間。<br><br>

無聊的本質是什麼？<br><br>
1.	很閒<br>
2.	很閒的本質是什麼？時間<br>
3.	時間是用來衡量「閒跟忙」<br>
4.	「閒跟忙」的最大區別是「急迫性」、「要做的事情太少/要做的事情太多」<br>
5.	無聊是「沒有急迫性」、「要做的事情太少」<br>
6.	從負面表達變成正面表達--> 悠閒 -->顯然有問題-->放棄<br><br>

Try 2: 	沒有興趣<br>
從負面表達變成正面表達--> 「討厭」正在面對的事<br>
「討厭」的背後是什麼？-->憤怒<br><br>

Try: 「要做的事情太少」的背後是什麼？<br>
--> 不知道自己要做什麼？<br>
→ 茫然<br>
→ 空洞<br>
Boring <br>
Boring a hole → 原地做重複的事情 → 變「空」<br><br>

不同方言有不同洞見。<br><br>

我習慣負面表述。<br><br>

一個在怎麼有道理的話，都可以推翻它。就是問題化。原因是，世界上沒有真理。真理是還沒有被推翻的假設。因此「問題化(problematization)」可以幫助我們練習自己的多元觀點。<br><br>



<h3 class="article-h1-color">素食</h3><br>

那我們該相信什麼？<br><br>

相信要有邏輯嗎？<br><br>

相信 > 信仰 > 一種決定<br><br>

行動，是指捍衛信仰嗎?<br><br>

政治人物在觀點上不同，但私下可能很好。<br><br>

怡人 ：吃素的人很多 ，跟特別的人很多<br>
→ 肉邊素<br>
→ 落葉素<br>
d→ 吃素的人裡，比你特別的人也很多<br><br>

→ 食氣者<br>
→ 就算最特別的吃素者，也沒有食氣者特別<br>
進一步問題化：吃空氣算是吃素嗎？<br>
→ 從「減少殺生」的目的來說，吃空氣當然算吃素<br>
進一步問題化：空氣裡有微生物，所以吃空氣也會殺生<br>
→ 所以世界上根本沒有吃素這件事<br>
進一步問題化：「減少殺生」的目的是避免生命被殺的「痛苦」，而不是生命被殺<br><br>

→ Ａ你怎麼知道植物、微生物被殺沒有痛苦？<br><br>

→ Ｂ你怎麼知道動植物不想要被吃？<br>
證據：植物目前已證實，被蟲吃食，會散發某個激素（氣味），會加強自己防禦力<br>
進一步問題化：像芬芳的花朵，有香氣的果實<br><br>

→ Ｃ你知道「殺生」可以幫助「生命」的成長嗎？<br>
證據：<br>
•	果實的香味<br>
•	疏林、疏果、小規模的森林火災<br>
•	魚苗、精子都是透過「殺生」來幫助「生命」的成長<br><br>


<h3 class="article-h1-color">素食</h3><br>

世界上沒有「真理」<br>
所謂的「真理」可能只是一個還沒有被推翻的「假設」<br>
因此「問題化(problematization)」找到邏輯上的問題，可以幫助我們練習自己的多元觀點<br><br>

那這樣我們該「相信」什麼？<br>
→ 「相信」需要符合「邏輯」嗎？<br>
不需要，因為相信是一種信仰，只需要主觀的「決定」就可以了。<br>
舉例：我該支持私生活不檢的政治人物嗎？<br>
包括「反核」，也是一種信仰<br><br>
我們在社會上有扮演的角色，會讓我們必須去有立場。這時候光是看到不同可能不夠。行動就是「捍衛信仰」的外在表現，不代表我不能接受信仰跟我不同的人。<br>
•	王安石和蘇東坡<br>
•	韓國瑜跟陳其邁<br>
•	警察跟黑道<br>
•	管理階層跟工會高層<br><br>

→ 面對跟自己價值觀不同的人怎麼辦？<br><br>
看懂不同的觀點，透過「問題化(problematization)」找到邏輯上的問題，可以看懂多元觀點就夠了。<br><br>

黑道跟白道，都認為是在保護人。<br>
不同黨派政治人物，都認為自己是在做對的事。<br>
反核不反核，都認為是在保護地球。<br>
台獨與親中，都認為是為這塊土地上的人好。<br><br>


<h3 class="article-h1-color">跟極端的距離更遠</h3><br>

跟極端反核的距離比較擁核的溫和者遠。<br><br>

體罰跟愛的教育，都是為了孩子好。<br><br>

作為反核人士，我有沒有意識到我跟「溫和的擁核者」比較接近，跟「極端的反核者」比較遙遠？<br>
→ 我作為反核人士跟「溫和的擁核者」共同點是什麼？<br>
→ 「愛地球」<br>
找到彼此的共同點。<br><br>

台灣人特別會找到別人的問題，要找到共同點。<br><br>

親戚每一句都是在找問題。<br><br>

英文的yes、but，等於no<br><br>

問題化，是找到邏輯上的問題。一直到你覺得無聊為止。<br><br>

作為邏輯練習，應該要可以一直推翻為止。<br><br>


觀察：說話者說....<br>
預設：說話者認為....<br>
拆解：這句話的「零件」之間邏輯有哪裡「壞掉」？（電風扇發出怪聲音）<br>
深化：說話者是一種什麼「類型」的人？<br>
概念化：這句話要說的「本質」變成一個詞是什麼？<br>
問題化：我要如何用邏輯推翻這句話，幫助對方看到自己的盲點？<br><br>

你怎麼知道植物、微生物被殺沒有痛苦？<br>
觀察：說話者說，植物有看不見的七情六慾。<br>
預設：說話者認為，有生理反應，就是有感覺。<br>
拆解：生理反應不等於感覺。<br>
深化：說話者很有同理心。<br>
概念化：同理心 > 大愛 > 在乎<br>
問題化：人有生理反應，不代表有七情六慾、有感覺。<br><br>

<h3 class="article-h1-color">當個白目</h3><br>

沒有信仰的人，隨時會被拉走，每個人的信仰都覺得有道理，部分認同。<br><br>

討論蘑菇，台灣人會想著，知道要幹嘛？<br><br>

為什麼有趣？我喜歡學新東西，沒要幹嘛。<br><br>

好奇，在台灣教育制度，很早就被扼殺。不能為不知道的東西學習。不會教，幹嘛學？不會考，幹嘛學？學這個有什麼用？<br><br>

好奇、想要知道，本身就是個很好的理由。<br><br>

邏輯只有一套，觀點、信仰可以不一樣。<br><br>

要當白目，地圓說。<br><br>

華人說：識時務者為俊傑<br>
