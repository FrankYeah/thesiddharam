想要合夥/創業，要學會當壞人

分享四則扎心案例：最難課題是人


文 / 西打藍 Siddharam

創業 / 合夥最高光時刻，是找到夥伴合作、外包，大家開好幾小時會議，靈感噴發，歡笑聲不斷，仿佛要創辦多麼偉大、突破式的產品 / 服務。

但是那天，即是創業過程最高光的日子，接下來面臨的人際實務面，會是一連串不舒服的開始：

創業夥伴工作成效不好，怎麼辦？
自己喪失合夥熱情，還繼續嗎？
外包夥伴被客戶嫌棄，你怎麼做？
滿心期待的外包成果不如預期，對方是你讀者 / 朋友怎麼辦？

人，可以說是創業 / 合夥過程最寶貴、也是最難解的一道題。我在接案 / 合夥 / 創業路上，痛苦了好多次。

下方來分享自己四個扎心案例：

創業夥伴工作成效不好，怎麼辦？


我曾有一個商品買賣的合夥案，原訂好要進貨三類品項，但是負責訂購的夥伴始終只採購一款，其他兩款拖延許久。

接著，他突然跑去接其他工作，無法投入時間在合夥事業上，於是進度停滯。我當時選擇靜觀其變，因為創業是他發起，我只是協助者。

幾個月後，夥伴說要重新復工，準備進貨其他品項，我則猶豫是否要離開，還是繼續參與。

創業常有說變就變的時刻，不只夥伴如此，我也會。例如下一個例子：

自己喪失合夥熱情，還繼續嗎？


我曾有項合作會進行群眾集資，我的目標是參與、體驗集資過程。

但是我發現，自己被其他更喜歡的事情佔據，只做合夥該做的事，投入時間少。很快被夥伴察覺了。

我們討論後，夥伴讓我暫停手邊工作，省思是否繼續。

我從這則經驗中學到：當沒動力時，不該反省拖延症，而要省思真的喜歡這項工作嗎？還是為做而做？

這件事後，我開始謹慎審視合作邀約，是真正感興趣、願意長遠發展的合作，才答應。

下兩個例子，來聊聊外包難題：

外包夥伴被客戶嫌棄，你怎麼做？


我曾找一名設計師合作網站，我們配合過多次。

而客戶是我好友，原以為合作會很順利，想不到讓我好痛苦。

當時設計師完成稿件後，我蠻喜歡，而客戶想修改，我和設計師則積極配合。

想不到不只修改一兩次，客戶前後修改七八次，並且要求一兩天要修改完成。

導致設計師、客戶兩人發生不愉快，客戶認為設計師很不優，我夾在中間好難做人，只好分別私下通電話，尋求和平結案方式。

這裡沒有誰對誰錯，而是認知不同。客戶認為外包要像員工隨時待命，但是接案邏輯並非如此。

雖然最後和平結案，但我始終有陰影，覺得和人溝通真難。

而當我是客戶本人，難道就不會有問題嗎？一樣會。

外包成果不如預期，怎麼辦？


我前陣子在找配合插畫師，原先有兩人投履歷，但都不適合。

直到我被第三位插畫師作品集打動，而決定外包。同時，他還是我的讀者。

我們在溝通風格時，可以看出他有理有據安排細節與流程，讓我很安心。

但是，直到看見他第一次提案成果，我心就涼了大半。不只是我，我的設計師夥伴也不滿意。

這時，我該請他修改？還是直接換人？

如果修改後又不滿意，怎麼辦？換人，會不會傷他自尊心？尤其他還是自己的讀者。

這些種種抉擇，是我時刻面臨的兩難。

老闆也想當好人，但是真的可以嗎？


我常聽人說：「如果我當老闆，一定會...」。聽到類似話語時，我總會苦笑心想：「誰不想當好人？」

當外包、夥伴工作成果不如預期，損失的是自己的品牌名譽、實實在在的金錢、客源。

要賭一把，相信夥伴等他改進？還是拆夥換人，破壞感情？

我相信有人會說：「創業可以不請人呀？」

但是人一天最多只能工作 16 小時，只靠自己拼搏不實際，況且客戶、客人都是人，「人」是無法逃避的命題。

如果只想做好事，無法狠心做決定的創業者，受傷害的會是自己。

我常認為，硬實力固然重要，但是創業家更該努力培養的是領導、溝通、業務、行銷等，與「人」直接相關的軟實力。

要扮黑臉，需要強大的內心，有讓人討厭的勇氣。創業最高光的時刻過後，接下來的多半是難言隱的苦澀。
