公婆家很傳統，未來相處不好怎麼辦？

對談分享


文 / 西打藍 Siddharam

近期陸續和幾位有煩惱、兩難的人，透過有紀律的思考對談。

這回是一對有家庭煩惱的男女情侶，女方提問：「公婆家很傳統，未來相處不好怎麼辦？」

從提問中，不難看出女方是個擔憂、有道德感、會妥協的人。

和女方確認性格無誤後，我們一起討論出幾個面對方式：

1. 女方維持預設作法，自己與公婆溝通。

2. 由男方出面解決。

3. 公婆改變習慣。

4. 結束婚姻關係。

透過觀察與預設，我們仔細檢查每個選項：

選項一，由女方溝通的結果，常會是傳統公婆堅持己見，因而產生衝突，或是女方無奈妥協。甚至導致，女方未來會成為新一位傳統婆婆。

選項二，自己的家人自己處理，這是有共識的最佳選項。困難的是，男方多會以「我們家就是這樣」、「不用理他們」，作為處理方式。

這是男方慣用的解法，卻沒有看見女方是個有道德感，希望家庭和睦的人。多數男生，包含以前的我，多是以這般冷處理和家人溝通。

選項三，這是最困難的方式。我們一致認為，想要公婆改變，不如改變自己。

選項四，既然男方不願意溝通、公婆不會改變、女方改變太痛苦，選擇結束關係，也不乏是個選擇。

我們討論結束後，女 / 男方的回饋如下：

女方回饋

1. 知道未來如何避免發生，也知道不讓男方處理會有什麼問題。

2. 知道男方的想法，但不知道對方會怎麼做。

3. 有被點醒的感覺，知道自己的性格，就是會成為傳統的媳婦。

4. 平常和朋友聊天，朋友都是給意見、告訴我怎麼做。但事情發生時，我根本不會這麼理性。

5. 問題變小了，覺得可以預防。

男方回饋

1. 喜歡和客觀、能包容的人討論事情。若是以「告訴」的方式進行，以後就不會再來了。

2. 喜歡塑造的討論氛圍。

3. 不喜歡問題沒有解決。

4. 問題沒辦法解決，怎麼辦？

確實，在對談中，我的角色是讓客戶知道自己的性格，引導他們梳理解決方法，而不是給出自己的意見。

對談過後，放下自己的角色，我向男方說：「你其實是知道解決方法的，不是嗎？」

男方想了想後，點了頭。

我說：「你曾問我『該怎麼和家人溝通』，表示你是一個孝順的人。你一定知道，只要自己不那麼孝順，就不會有問題了，不是嗎？」

最後，男方聽懂了，他一直是知道解決方法的。

而我也知道，他不會改變的。但，那有什麼問題嗎？

性格決定命運，也註定了這樣的人生會遇見什麼難題。

孝順與不孝順，都有自己的人生課題要面對。

若真要我給建議，我會說：「去面對、喜歡自己所選擇的人生困境吧！」
