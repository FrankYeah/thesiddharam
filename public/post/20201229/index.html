離職第四十四週

一場沒有成就感的演講


文 / 西打藍 Siddharam

週三接了一場大一演講，是一位通識老師找我代課演講。演講過程，有些空虛。

「這些學生平常就是這樣，不論上課或演講，他們不會互動，都在玩手機或聊天。」通識老師略帶歉意地說。

原先主題是「職場」，老師希望我聊聊工程師的工作現狀。然而這群學生才大一，職場離他們太遠，講工作細節挺沒意思。

和朋友討論後，決定以「人生不是單行道，當下的你擁有許多選擇」為主題，分享我當記者、轉職、成為自由工作者的契機，聊聊選擇與行動。

可惜演講過程，除了第一排學生會互動外，其餘學生多在玩手機或聊天。

我想，也不能怪他們。

這畢竟是一堂不得不上、為學分而存在的通識課，學費也不是他們出，能夠理解不專心上課的原因。

都過那麼多年，這樣的上課模式依舊不變，老師與學生相互消耗，究竟是為了什麼？

1，創業團隊網站修改

這回多是文案修改，較麻煩的是 oauth 註冊/登入，和一般處理 API 方式不同。

2，南美洲網站開發

這是每週最大的工作負擔，這週處理了 CSV 存檔、修改 Nuxt 登入跳頁設定等九項任務。

3，採訪室內設計公司

是久違的採訪，討論了不同室內設計風格差異。

4，新工作－人物傳記

接到一個新案子，明年初會採訪一位八十多歲長輩的一生經歷。

人物，一直是自己有興趣的主題，未來也想紀錄自己的家，先從別人家開始吧。

5，修改高遠文章

對方告知修改細節，調整後，應該能結案了。

6，演講

先前曾去另一所大學演講，是由社團學生邀請，聽課學生對演講主題興趣極大，過程中交流不斷，結束後也來聊天交流。

人的內在動機，果然才是做事的動力。

7，交換禮物

這週兩次交換禮物主題都挺有意思，一個是交換最喜歡的零食，一個是紅綠色的實用禮物。

8，《Sweet Home》

一直很喜歡殭屍類型影劇，《Sweet Home》的配樂好好聽，循環了好幾天《Warriors》這首歌。
