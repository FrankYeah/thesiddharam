絕望、成就感與人生意義

這幾天一次體會


文 / 西打藍 Siddharam

終於有時間好好寫字，來整體說說近況感想。

從過年後到上週末，經歷了每天起床就要立刻到電腦前報道，聽著基隆連著幾個月的雨聲，一路工作忙碌至半夜。

228 連假？沒有出門。記得那時，我和一位連假爬山的朋友說：好羨慕你。

好不容易，上週總算截完三項案子，陸續開心收款，身上剩下常態性合作與新文字案規劃。

總算，絕望的情緒過去了。在有餘裕的工作狀況下，成就感在日常中出現了。

成就感


近期換了台 Mac，讓陪伴五年的 win 電腦逐步退役。

我幾乎每天都在熟悉快捷鍵：螢幕截圖、大小中英切換、顯示隱藏檔、標點符號庫......

工程方面更麻煩，要重新安裝各工具：Brew、Git、Hugo、NVM、NODE、SSH-AGENT......

也幾乎每天在解決問題：

- 解決網站跨域問題，最後直接拉下後端 code 修改，用 docker 測試資料庫。

- 幫家人搬家，第一次設定小米監視器與多人分享。

- 啟動 Hugo （部落格網站生成器）失敗，研究一整晚，才發現是版本不同問題，Mac 更新版本過新。

每解決一項問題，滿足與成就感油然而生。

在工作之餘，這兩天也多了自由時間，可以列下「想做的事」，而非「必要工事」。

人生意義


昨天早上，一邊和學生在麥味登討論程式進度，一邊解網站 bug。解完後，下午和樂心書室經營者有約。她去年剛畢業，沒有選擇投入職場，而是選擇改造基隆老屋開書店。

我們聊了快四個小時，聊了寫作、接案、基隆，也和她分享近期正規劃的文字工作坊。

口乾舌燥，心底滿足。

今天回想著昨日慢慢交換想法的過程，發現在當下的時光，可以說就是人生的意義。

或者可以說，每一刻，都是人生意義的全部。無論是絕望，或是獲得成就感。

小故事


最後分享件寫文過程發生的小事。

突然有人用通訊軟體敲我，通知視窗中，我想不起來這個人是誰。

點開訊息才想起一件事。

有天搭車到台北，前方有位女生一直無法刷過悠遊卡，我看著她卡在入口好久，看起來緊張又心慌，於是把身上另一張悠遊卡給她，她跟我道謝並交換通訊軟體，約定改天還卡。

接著，我就忘記這件事了。直到好幾個禮拜後的今天，她密我才想起來。

真是喜歡這些未知與緣分。

差點忘了，人與人間每一次的交集，都是緣分啊。
