離職第一週

做了什麼事 | 什麼事情發生了


文 / 西打藍 Siddharam

離職之後


我還沒寫為什麼離職，因為還想沉澱一段時間再說。而今天有點感覺，想先記錄離職後發生的事，就開始吧。

懵懂的開始


從 2/21（五） 最後一天工作，一直到今天（2/27），我都很謹慎地對待每一日。如果以每月薪資除以 30 來算，每天至少有一千五百元的價值，要如何自己雇用自己，才能達到這個價碼呢？

在離職前，我就在 trello 寫下筆記，紀錄將來要寫的技術、思考的文章，以及離職後有哪些事可以做。而最明確希望做的，就是每天可以生產一篇部落格文章。

除了 trello 之外，還下載了 Tasks，它是一款待辦事項工具。它有兩項作用：第一是自我獎勵，讓自己在完成每一件待辦事項後打勾，得到成就感；第二則是記錄離職後每一天完成了什麼事。

至今為止，每天在睡覺前，都如期的完成一篇文章，並在隔天早上發表。

最後，還更新了部落格的封面底圖，謝謝御用設計師。

聯絡朋友


打開 trello，其中一項要做的事情，是聯絡朋友。

於是便敲了一位過去當記者時的同事，他從公司離職後，休息了近八個月。閒聊中，他分享了人生目標，還有對現實的挫敗，而在休息這麼長一段時間後，他才又有能量重新開始找工作。

另外聯絡的一位是高中好朋友，他剛提離職，他提到有創業的念頭，但還沒有太多想法，想多聊聊。因為很要好，這幾天也常碰面，有種彼此打氣的開心。

臨時小案子


由於加入了不少 FB 的外包社團，週二（2/25）早上，其中一個人密我，希望我幫他改版網站，是一個透過開源地圖取得路徑，來預估車資的網站。

在打開程式碼後，發現一個 HTML 頁面裡，竟然四處散落 script tag，而使用的 here WeGo 的 API 文檔資料也過少，便在當天回絕了這個小案子。

加入新團體


在離職前，我常到基隆的無用學堂蹭吃聊天，兩位主人是去年才開始在基隆經營空間。他們一邊兼職，一邊試著在空間中創造些什麼，我很喜歡他們。也在這裡遇見失散已久的妹妹。這是後話。

週二（2/25）中午，他們邀請我參與，舉辦九月份基隆活動的會議，由於尚未定案便不公布內容。除了我們外，還包括其他三個基隆在地的公民組織共同籌畫。

我們分了創作組、行銷組和行政組，我決定加入了需要找錢的行銷組，這是我最有興趣的部分，而日後的網站架設也會由我負責。

之後的每周二都會開會，將來也會為這個鍾愛的團體寫一篇文章。

第一件外包案


在週三（2/26）早上，朋友介紹一件外包案給我，內容是網站全面改版、轉換後台系統（CMS）、文章搬移以及網址更新。

其中的後台轉換及文章搬移讓我焦慮，因為先前沒做過這件事，但也正好是個練習機會，因為太多的案子需要用到後台，WordPress 的學習本就排在離職的待辦事項中。

而在談了時程和價碼後，也送出了報價單，就看接下來會怎麼合作。如果合作順利的話，或許原定三個月不找工作的打算，有機會延長了。

至於談妥的價碼和合作單位就先不公布，如果三個月過後順利完成，我會在談後續。

重機駕照


家裡有一台許多未騎的重型機車，是不須打檔的黃牌重機，但一直沒有動力去考，而在聯絡高中好朋友後，他也正想考重機駕照，索性就一塊報名，我們在週三（2/26）晚上完成了體檢。

好友出書了


好友 amazing 在年前聚餐時告訴我們，他即將出書了！得知他的書本內容，是和離職後的省思有關，就一直很期待它的出版。而就在上周終於開放訂購，我也額外買了一本給高中好朋友讀。希望這個連假可以好好讀完它。

書名：有一種工作，叫生活。
出版社：遠流。
作者：曾彥菁Amazing。

結語


一切的一切似乎都正發展著。截至目前，心理仍充滿著踏實感。

我想，這就是為自己工作的感覺吧。
