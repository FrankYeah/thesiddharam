身處網路業的焦慮，你也有嗎？

無止盡的網路新名詞


文 / 西打藍 Siddharam

身處網路業，對每年推陳出新的名詞與技術，很難不感到焦慮。

記得我大學時，FB 開始興盛， Web 技術逐步演進。沒多久 Youtube 出現，影音領域開始茁壯，APP 也隨著手機發展成為顯學。

出社會幾年後，朋友漸漸湧入 IG，國外則是 Twitter 當道，使用 FB 的年齡層越趨上升。

除了社群，早期發展的部落格越發穩定，熟捻 SEO 便能掌握流量，聯盟行銷成為賺錢良方，電商產業也日益興盛。而網路業的發展不止步於此。

搶佔你的注意力


4G 網路穩定後，直播平台、抖音、短影片相繼出現，網路行銷再度變形，從業人員苦上加苦，我也在當時考慮轉行工程師。

順利轉職後，我所熟悉的 Web 前端技術，逐漸被迭代出新的 iOS、Android 系統追上，Python、AI 技術也一度是追捧熱潮。

我們當然不會忘記，那些轉瞬即逝的語音平台、熱度依舊的 Podcast，以及個人品牌愛用的郵件行銷。

即使是實體店面，也沒錯過網路行銷的大戲。Google 商家的研究必不可少，實境解謎 RPG 也仍蔚為流行。

而今盛頭正勁的，必然是區塊鏈，由它延伸的 Telegram、Discord 社群成為幣圈消息管道。FB 已然成為資訊末站，僅稍贏 OTT 新聞媒體。

虛擬貨幣的延伸不只如此，近期出現的 NFT 藏品、智慧合約，已經成為新創公司的專案項目。

我們會是其中的參與者、創造者，還是旁觀者？

要逐利，或求穩？


當然，無論你的角色定位，有技術、有能力永遠有飯吃。

新的虛擬貨幣平台，還是需要架設 Web 網站，而 iOS、Android APP 依舊是交易介面。

更別說掌管資料庫的後端工程師，每間網路公司都有增刪改查與更複雜的需求。各個社群平台，經營者也都得斟酌投入。

不同的是，流量與轉換紅利的差異。例如有些行業別特別適合短影音，幾天前，就有帥哥在燒烤店幫客人烤肉，放到抖音後，店家瞬間爆紅的案例。

身處網路業的我們，該何去何從？

快/慢世界，何為真？


說實話，我不知道。就連上述分析正確與否，我也沒把握。

我確定的是，對技術的需求依然存在。軟體工程師永遠供不應求，真正厲害的行銷人才更是稀缺。

我前幾天離開電腦桌前，到嘉義住了一週，尋訪這座城市的特色空間。我認識許多在地青年，他們生活的重心與我全然不同。

青年關心舊物的存續，老屋、花磚、經典古著，他們慢慢、好好生活著。

我們兩者，一快一慢，追新逐舊。

我突然思付起：這一世代的網路業者，離世以後，會留下什麼痕跡？

粉專、IG、短影音，留住什麼了嗎？

還是不再更新後，就消失了呢？

我常常想起一位朋友，他在臉書興盛時短暫參與後，就離開人世。

當時的他，能想像現在網路的百花齊放嗎？

靠學習緩解焦慮


說到這，我也不是這麼悲觀與擔憂，是對身處網路世代有感而發。

我也發現最紅的網路課程，必然是現代人最深的焦慮。

我們都擔心被世代淘汰，並且極可能在某一天成真。

但，我們要為那一天而無止盡焦慮嗎？

我突然想到自己喜歡的蘇格拉底，他連網路是什麼都不知道，我仍然喜歡他，想向他學習。

思及至此，也夠了，該來去面對手上的案子，這才是當下重要的事。
